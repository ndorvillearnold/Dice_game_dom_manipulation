The winner is displayed on the screen along with the dice rolls. This is a fun way for my kids to learn about randomness and fairness in a simple way. The main goal is to ensure that only one player rolls at a time. ## How it Works 1. **Randomizing Numbers**: A function is created to generate random numbers for the dice rolls. 2. **Displaying Dice Rolls**: Both players' dice rolls are displayed using the `document.getElementById()` method. 3. **Determine Winner**: The game compares the dice rolls and displays who wins. ## Game Flow 1. Player 1 rolls the dice, generating a random number between 1 and 6. 2. Player 2 rolls their dice. 3. The game compares the dice rolls. 4. The player with the higher roll is declared the winner, and the other player will do the dishes! ## What I Learned - **DOM Manipulation**: I learned how to use `document.getElementById()` to grab elements from the HTML and update their content dynamically.
